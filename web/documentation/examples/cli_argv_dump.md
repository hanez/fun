What it shows
- Prints argv/argc handling to demonstrate CLI arguments parsing at a low level.

How to run
- export FUN_LIB_DIR="./lib"
- ./build_debug/fun examples/cli_argv_dump.fun --foo bar 123
- Or: fun examples/cli_argv_dump.fun --foo bar 123
